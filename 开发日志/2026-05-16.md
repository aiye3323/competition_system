### 十三、OCR 识别增强

| 改进 | 说明 |
|------|------|
| 逐行标签匹配 | 支持标签与值跨行分离的证书格式 |
| 日期解析统一 | `extractDate()` 支持 `-`、`/`、`.`、年月日 四种格式，自动补零 |
| 登记号多层兜底 | 标签匹配 → `\d{4}SR\d{5,}` → `\d{4}5R\d{5,}` → 登记行长数字 → 泛模式 |
| 标签去空格 | `matchLabel` 先去除空格再匹配，防 OCR 将"登记号"识别为"登记 号" |
| 新增字段 | 竞赛：指导教师、参赛选手；项目：指导教师、立项人员 |
| 前端改用 fileId | `recognizeOcrById(fileId)` 替代 DOM querSelector，稳定可靠 |
