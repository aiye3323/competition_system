# 竞赛科研管理系统

## 项目简介
本系统用于高校竞赛成果和科研成果的申报、审核、统计管理。支持学生/教师提交成果，科研秘书一审，学院领导终审的完整工作流。

## 技术栈
- **后端：** Spring Boot 3.2.5 + Spring Security + Spring Data JPA
- **前端：** Vue 3 + Vite + Element Plus
- **数据库：** MySQL 8.0
- **构建工具：** Maven（后端）/ npm（前端）

## 启动方式

### 后端（IDEA）
1. File → Open → 选择 `backend/pom.xml` → Open as Project
2. 等待 Maven 依赖下载完成（右下角进度条消失）
3. 右键 `CompetitionApplication.java` → Run 'CompetitionApplication'
4. 默认地址：`http://localhost:8080`
5. 测试接口：`http://localhost:8080/api/test`

### 前端（VS Code 终端）
1. File → Open Folder → 选择 `frontend/`
2. 终端执行：`npm run dev`
3. 默认地址：`http://localhost:5173`

### 数据库
- 数据库名：`competition_db`
- 连接信息：`localhost:3306`，`root` / `123456`
- 字符集：`utf8mb4`，排序规则：`utf8mb4_unicode_ci`
